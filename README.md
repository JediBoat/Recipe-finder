# Recipe-finder

## Setup Requirements for the Database
To use our project, you will need to set up your database and create your own tables with the prepared queries since ours is inaccessible. 
All of the queries for database creation and data insertion should be in the database.py file, which is in the Database Python files.

## Setup Requirements for the PHP pages

To run the PHP components locally:
1. Install PHP: https://www.php.net/downloads.php#/
2. Launch a local PHP server
3. Run `search.inc.php` to access the main page
